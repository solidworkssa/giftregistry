# GiftRegistry

On-chain gift registry and contribution tracking.

## Features
- Multi-chain support (Base & Stacks)
- Secure Contract Logic
- React/Next.js Frontend

## Contracts
- Base: `contracts/base/src/GiftRegistry.sol`
- Stacks: `contracts/stacks/contracts/giftregistry.clar`

## Setup
```bash
pnpm install
pnpm dev
```
